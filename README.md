# Student Feedback - Simple JS project

This is a tiny demo project for the student feedback form assignment. It contains a minimal `index.js` which stores a couple of sample feedback entries and prints a summary.

How to run:

1. Install Node.js (if not installed).
2. From project folder run `npm start`.

This repository is intended to be committed and pushed to your GitHub remote and then cloned to another folder to verify synchronization.
